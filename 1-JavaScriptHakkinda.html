Javascript, betik(script) dilidir. (java ile karıştırmayınız java server side bir programlama dilidir.) Derlenmeye ihtiyaç yoktur tarayıcı tarafından yorumlanır. Düz html sayfalarına dinamiklik katmak için kullanılır.

Java Script Kodlarının yazılması için

  • Elementin içine doğrudan yazılabilir.
  • Head bloklarının arasında script tagleri arasında yazılabilir
  • Sayfanın body bitişinden önce script tagleri arasına yazılır.
  • External javascript dosyası içinde.
Sayfanın açılış hızını yavaşlatmaması için body bloğunun bitişinden hemen önce yazılması önerilir.

1.Element içine tanımlama

2.Head bloğuna script tagi ile tanımlama.

3.Body'nin üzerinde script tagleri ile tanımlama.
